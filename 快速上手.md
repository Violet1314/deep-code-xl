# 快速上手

---

## 1. 安装

打开 PowerShell，执行：

```powershell
npm install -g deep-code-xl
```

看到 `✓ Deep Code 安装完成` 即安装成功。

> **安装太慢？** 先切国内镜像：
> ```powershell
> npm config set registry https://registry.npmmirror.com
> ```
> 再执行安装命令。

---

## 2. 首次配置

```powershell
deep-code
```

首次启动会自动进入配置引导：

```
=== Deep Code Login ===

Select provider preset:
  [1] DeepSeek 官方 (https://api.deepseek.com)
  [2] OpenAI (https://api.openai.com/v1)
  [3] Qwen (https://dashscope.aliyuncs.com/compatible-mode/v1)
  [4] GLM (https://open.bigmodel.cn/api/paas/v4)
  [5] Kimi (https://api.moonshot.cn/v1)
  [0] Custom (手动输入地址)

Choice (default 1): 1
Base URL [https://api.deepseek.com]:
API Key: *****
```

- 输入 API Key 时字符不显示（安全设计），输入完按回车即可
- 配置保存后下次启动直接使用，无需重新配置

---

## 3. 开始使用

```powershell
deep-code
```

进入交互界面后，直接用自然语言描述需求：

```

```

```
> 把 config.json 中的 active_provider 改成 deepseek
```

```
> 编译项目并运行所有测试
```

```
> 解释一下 app/app.go 中的 ensureConfig 函数做了什么
```

---

## 4. 常用命令

在交互界面中输入 `/` 开头的命令：

| 命令 | 作用 |
|------|------|
| `/help` | 查看所有命令 |
| `/model` | 切换模型 |
| `/connect` | 重新配置 API |
| `/theme` | 切换主题 |
| `/clear` | 清空当前对话 |
| `/quit` | 退出 |

---

## 5. 主题切换

```powershell
# 查看可用主题
/theme

# 切换主题
/theme mono
```

当前可用主题：`default`、`mono`。

---

## 6. 自动更新

Deep Code 每次启动会自动检查 GitHub 是否有新版本：

- 有新版：自动下载 → 替换 → 重启（你会看到进度条）
- 已最新：无任何提示，直接进入

你永远使用最新版本，无需手动操作。
